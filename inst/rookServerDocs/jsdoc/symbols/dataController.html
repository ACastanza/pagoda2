dataController(repository)
Handles all data requests At the moment these will be simple callbacks to the backend Rook server. In the future this will be loaded from a local file or it can be requested from an external resource database (cross-site scripting security issues may arise). It can also cache requests.
Parameters:
repository
this is the data source currently only remote is supported this can alternatively be a local
